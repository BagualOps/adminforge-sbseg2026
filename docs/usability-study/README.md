# Usability study materials

Materials of the exploratory usability study reported in the AdminForge paper (5 Linux administrators, 9 tasks, moderated think-aloud sessions).

## Executed protocol (replication package)

- `GUIA_FACILITADOR.md` / `.docx`: facilitator guide (session setup, seeded drift on web-02, observation sheet, interview script)
- `ROTEIRO_PARTICIPANTE.md` / `.docx`: participant task script (Tasks 0-9, Alice scenario)
- `QUESTIONARIO.md`: all questionnaire instruments (profile, immediate post-task ratings, TAM-based post-test)
- `criar-formulario-google.gs`: generates the participant questionnaire as a Google Form
- `criar-formulario-consentimento.gs`: generates the consent form
- `lab/`: the Docker laboratory (three sshd containers, bootstrap and reset scripts)

## Kept for provenance, not part of the executed protocol

- `METODOLOGIA-OPCOES.md` / `.docx`: methodology decision record written BEFORE the study; it ranks candidate methodologies and mentions a 10-15 participant target. The executed study followed its Option 1 with 5 participants.
- `criar-formulario-facilitador.gs`: digital version of the facilitator observation sheet. Not used in the reported sessions; the facilitator logged observations manually.
- `referencias/`: background papers (includes SUS, which the final instrument does not use; the post-test uses TAM only, as documented in `QUESTIONARIO.md`).
